
monitor server —— logger 日志监控
===============

> V1.0.3
php_logger V1.0.2 在 V1.0.1 基础上做了进一步的改进，更新后主要特性：

 + 支持日志控制台打印
 + 支持日志文本存储
 + 支持日志MYSQL存储

> 运行环境要求 => 依赖库
pthread: pthread
boost：yum install boost-devel
mysql-connector: mysql-connector-c++-1.1.12-linux-glibc2.12-x86-64bit

## 使用

+ 运行监听所有客户端日志数据
+ 需要客户端开启server配置
